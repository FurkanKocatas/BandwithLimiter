# Build Instructions

## Prerequisites

### All Platforms
- CMake 3.16 or higher
- Qt6 (Core, Widgets modules)
- C++17 compatible compiler (GCC 7+, Clang 5+, MSVC 2017+)

### Linux
```bash
# Ubuntu/Debian
sudo apt-get install build-essential cmake qt6-base-dev qt6-base-dev-tools

# Fedora
sudo dnf install gcc-c++ cmake qt6-qtbase-devel
```

### Windows
- Install Qt6 from https://www.qt.io/download
- Install Visual Studio 2019 or later with C++ support
- Set Qt6_DIR environment variable or specify in CMake

### macOS
```bash
# Using Homebrew
brew install cmake qt@6
```

## Building

### Linux
```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j$(nproc)
```

### Windows
```bash
mkdir build && cd build
cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```

### macOS
```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j$(sysctl -n hw.ncpu)
```

## Running

### Linux/macOS
**Note**: Requires root/sudo privileges for network throttling
```bash
sudo ./BandwidthThrottler
```

### Windows
Run as Administrator for network throttling functionality.

## Troubleshooting

### Qt6 not found
Specify Qt6 path:
```bash
cmake .. -DQt6_DIR=/path/to/qt6/lib/cmake/Qt6
```

### Missing UI header
The `ui_MainWindow.h` file is auto-generated by Qt's UIC tool. If missing, ensure:
- CMakeLists.txt has `set(CMAKE_AUTOUIC ON)`
- MainWindow.ui exists in src/
- Qt6 is properly configured

### Permission errors (Linux/macOS)
Network throttling requires root access. Ensure:
- Running with sudo/root privileges
- cgroups are properly mounted (Linux)
- pfctl is available (macOS)
